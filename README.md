# 부르지 못한 말 (Hong Gildong)

허균 「홍길동전」 — 김유정 고쳐씀(1935 『新兒童』, 퍼블릭도메인) 오마주 — 자객의 밤, 홍판서 댁의 서자.
**가로 화면**의 다섯 구획(별당 → 도적 소굴 → 해인사 → 감영 → 대궐 앞뜰)을 지나 아홉 자물쇠와 책 사이퍼를 풀어라.
쇠사슬에 끌려간 것 또한 짚으로 빚은 제웅 — 그리고 마지막 자물쇠가 묻는 것은 보물이 아니라, **평생 부르지 못한 말**이다.
